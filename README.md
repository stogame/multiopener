Multiopener
===========

starts multiple applications

Usage
-------

For each application add the absolute path you into the `apps.txt` file.

**Example:** 

    C:\Path\To\iTunes.exe
    E:\Path\To\Another\Application.exe
    .....

Just execute the `start.bat` file.
